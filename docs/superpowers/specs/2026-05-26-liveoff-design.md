# LiveOFF — Спецификация дизайна MVP

## 1. Обзор

**LiveOFF** — мобильное Android-приложение: сборник советов по выживанию и безопасности для мирных жителей. Работает на 90% без интернета. Контент включает действия при воздушных тревогах, ядерной/химической угрозе, первую помощь, навыки выживания в природе и бытовые навыки при отключении инфраструктуры.

**Пакет:** `com.lefesafety.liveoff`

## 2. Целевая аудитория

Мирные жители, которым нужны практические знания безопасности и выживания в условиях обстрелов, а также бытовые навыки на случай отключения инфраструктуры.

## 3. Технический стек

- **Язык:** Kotlin 2.0+
- **UI:** Jetpack Compose (BOM)
- **Архитектура:** Clean Architecture (Data → Domain → UI)
- **Навигация:** Compose Navigation
- **DI:** Hilt
- **БД:** Room + FTS4 (полнотекстовый поиск)
- **Настройки:** DataStore Preferences
- **Асинхронность:** Kotlin Coroutines + Flow
- **JSON-парсинг:** Kotlinx Serialization
- **Контент Markdown:** Compose Markdown / Richtext
- **Фонарик:** CameraX (torch mode)
- **Звук:** MediaPlayer
- **SMS/Звонок:** SmsManager / Intent
- **minSdk:** 24, **targetSdk:** 36

**Не включено в MVP:** Firebase, аналитика, Retrofit, карты, push-уведомления, загрузка пакетов с сервера.

## 4. Визуальный стиль

Прагматичный/тёмный: тёмный фон (`#1e2126`), карточки `#2a2e35`, цветные акценты по категориям. Экономит батарею на AMOLED. Высокий контраст для чтения в стрессе. Ночной режим в настройках.

Цвета категорий:
- Огонь и тепло: `#ff8c00` (оранжевый)
- Вода и питание: `#3498db` (синий)
- Первая помощь: `#e74c3c` (красный)
- Радиация/Химия/Воздушные атаки: `#e74c3c` (красный)
- Навигация и инструменты: `#2ecc71` (зелёный)
- Эвакуация: `#2ecc71` (зелёный)
- SOS: `#ff2200` (ярко-красный)
- Морзе: `#a29bfe` (фиолетовый)

## 5. Структура экранов

### 5.1. Главный экран
- Быстрые ссылки: SOS, Чек-листы, Карточки, Поиск, Избранное, Морзе, Настройки

### 5.2. SOS-экран
Доступен с любого экрана через Floating Action Button.
Крупные кнопки:
- **Сигнал** — громкий звуковой сигнал (3 цикла)
- **Фонарик** — стробоскоп / фиксированный свет / SOS-морзе
- **Чек-лист эвакуации** — быстрый список: документы, аптечка, вода, одежда, зарядка, ключи
- **Мои контакты** — кнопка позвонить / отправить SMS (локально)
- **Первая помощь** — быстрый доступ к инструкциям
- **Тихий режим** — экран затемняется, звук выключен, только вибрация

### 5.3. Чек-листы
- Список категорий → список чек-листов внутри категории → чек-лист с интерактивными галочками
- Прогресс чек-листа сохраняется локально

### 5.4. Карточки (инструкции)
- Список карточек с тегами (время, сложность, инструменты)
- Детальный вид: **краткий режим** по умолчанию (4-6 шагов) + кнопка "Подробнее" разворачивает полную инструкцию в Markdown
- Кнопки: "В избранное", "Прочитано"
- Внизу: предупреждения, ссылка на экстренные контакты

### 5.5. Поиск
- Полнотекстовый поиск через FTS4
- Фильтры: категория, сложность, офлайн/онлайн
- Быстрый фильтр "Только офлайн"

### 5.6. Азбука Морзе
- **Таблица символов** — русский алфавит + цифры 0-9, по тапу воспроизведение звуком/вибрацией
- **Тренажёр** — приложение показывает букву, пользователь тапает точки/тире, обратная связь
- **Передатчик** — ввод текста → Морзе → воспроизведение фонариком / звуком / вибрацией (выбор)
- **Быстрые сигналы** — SOS, "Помогите", "Да", "Нет" одной кнопкой
- Кнопка "SOS фонариком" также доступна на SOS-экране

### 5.7. Контент
- Весь контент встроен в APK (`assets/content/base.json`)
- Обновление контента — через обновление приложения в Google Play
- Экран "Пакеты контента" убран из MVP

### 5.8. Избранное / Мои заметки
- Сохранённые карточки и чек-листы
- Персональные текстовые заметки (локально)

### 5.9. Настройки
- Язык, единицы измерения
- Ночной режим, отключить анимации
- Автозагрузка пакетов по Wi-Fi
- Конфиденциальность (нет регистрации по умолчанию)

## 7. Модель данных (Room)

### Category
| Поле | Тип | Описание |
|------|-----|----------|
| id | String (PK) | Slug: "fire", "water", ... |
| name | String | "Огонь и тепло" |
| icon | String | Имя ресурса иконки |
| accentColor | String | Hex-цвет: #ff8c00 |
| sortOrder | Int | Порядок отображения |

### Card
| Поле | Тип | Описание |
|------|-----|----------|
| id | String (PK) | UUID |
| categoryId | String (FK) | → Category |
| title | String | Название карточки |
| briefSteps | List\<String\> | Краткие шаги (4-6) |
| detailedContent | String | Markdown — подробная инструкция |
| warnings | List\<String\> | Предупреждения |
| difficulty | Enum | EASY, MEDIUM, HARD |
| estimatedTime | String | "5 мин" |
| tools | List\<String\> | Необходимые инструменты |
| tags | List\<String\> | Теги для фильтрации |
| packageId | String | "base" или ID пакета |
| version | Int | Версия контента |

### Checklist
| Поле | Тип | Описание |
|------|-----|----------|
| id | String (PK) | UUID |
| categoryId | String (FK) | → Category |
| title | String | Название чек-листа |
| items | List\<ChecklistItem\> | Пункты (text + sortOrder) |
| packageId | String | ID пакета |

### Favorite
| Поле | Тип | Описание |
|------|-----|----------|
| id | Long (PK, auto) | |
| contentId | String | ID карточки или чек-листа |
| contentType | Enum | CARD / CHECKLIST |
| addedAt | Long | Timestamp |

### UserNote
| Поле | Тип | Описание |
|------|-----|----------|
| id | Long (PK, auto) | |
| contentId | String | К чему привязана заметка |
| text | String | Текст заметки |
| updatedAt | Long | Timestamp |

### ChecklistProgress
| Поле | Тип | Описание |
|------|-----|----------|
| checklistId | String (PK) | → Checklist |
| itemIndex | Int (PK) | Индекс пункта |
| isChecked | Boolean | Отмечен ли |

### ReadStatus
| Поле | Тип | Описание |
|------|-----|----------|
| cardId | String (PK) | → Card |
| readAt | Long | Timestamp |

### CardFts (FTS4)
Виртуальная таблица для полнотекстового поиска по: title, briefSteps (склеенные), detailedContent.

## 10. Азбука Морзе

### MorseCodec
- Поддержка русского алфавита (А-Я), цифр (0-9), базовых знаков препинания
- Двусторонняя конвертация: текст → Морзе, Морзе → текст
- Таблица кодов как константа в коде

### MorsePlayer
- Три режима воспроизведения: фонарик (CameraX torch), звук (MediaPlayer/ToneGenerator), вибрация
- Настраиваемая скорость (WPM)
- Таймеры: точка = 1 ед., тире = 3 ед., пауза между символами = 3 ед., пауза между словами = 7 ед.

### Экраны
- **Таблица символов** — сетка русского алфавита + цифры, тап → воспроизведение
- **Тренажёр** — рандомная буква, пользователь вводит точки/тире, обратная связь
- **Передатчик** — текстовое поле → выбор режима → воспроизведение
- **Быстрые сигналы** — SOS, Помогите, Да, Нет — одной кнопкой

## 11. Офлайн-стратегия

- Весь контент (все 17 карточек/чек-листов + Морзе) встроен в APK в `assets/content/base.json`
- При первом запуске `ContentImporter` парсит JSON и заполняет Room
- FTS4 индекс строится при импорте
- Обновление контента — через обновление приложения в Google Play (новая версия APK)
- Все пользовательские данные (избранное, заметки, прогресс) — только локально
- Приложение работает 100% офлайн, интернет нужен только для SOS (SMS/звонок через сотовую связь)

## 12. Навигация

- SOS FAB — на каждом экране, один тап → SOS-экран
- Главный экран → категории/экраны через быстрые ссылки
- Глубина: Главная → Категория → Список → Детали (карточка или чек-лист)
- Back-навигация стандартная Android
- Без bottom navigation — SOS FAB важнее стандартных табов

## 13. Ограничения и допущения

- Нет серверного API — весь контент в APK, обновляется через Google Play
- Нет регистрации / аккаунтов — полная приватность
- Нет синхронизации между устройствами
- Контент на русском языке (MVP)
- SMS/звонок на SOS-экране используют системные Intent — работают при наличии сотовой связи
